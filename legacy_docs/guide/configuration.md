# Configuration

[Getting Started](../getting-started/installation.md) · [Guide](usage.md) · [Reference](../reference/commands.md)

Configure Jerd interactively from within an initialized directory.

```bash
jerd config
```

This opens an interactive configuration menu where you can update preferences without manually editing files.

## What you can configure

- Editor selection (opens entries in your preferred editor)
- Templates (JSON-based)
- Themes (Cozy, Neon, Minimal)
- Other preferences exposed by the CLI
