# Quick Start

[Getting Started](installation.md) · [Guide](../guide/usage.md) · [Reference](../reference/commands.md)

Jerd is built for “start writing now” workflows.

## 1) Initialize

```bash
jerd init
```

## 2) Create today’s entry

```bash
jerd new
```

## 3) Open an entry

```bash
jerd open today
```

That’s all you need to begin.
